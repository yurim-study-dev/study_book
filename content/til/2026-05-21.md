## 2. switch 표현식 (Switch Expression)

Java 12에서 도입된 새로운 switch 문 형태입니다.

기존 switch문의 단점을 보완합니다:
- break 키워드가 불필요
- 화살표 연산자 (->) 사용
- 값을 직접 반환할 수 있음

### 기본 형식

```java
String result = switch (변수) {
    case 값1 -> "결과1";
    case 값2 -> "결과2";
    default -> "기본값";
};
System.out.println(result);
```

### 기본 switch문과의 비교

기본 switch문 (break 필수):
```java
String role = "ADMIN";
switch (role) {
    case "ADMIN":
        System.out.println("관리자 권한");
        break;
    case "USER":
        System.out.println("일반 사용자 권한");
        break;
    default:
        System.out.println("권한 없음");
        break;
}
```

switch 표현식 (break 불필요):
```java
String role = "ADMIN";
String result = switch (role) {
    case "ADMIN" -> "관리자 권한";
    case "USER" -> "일반 사용자 권한";
    default -> "권한 없음";
};
System.out.println(result);
```

## 3. switch 표현식의 주요 기능

### 다중 조건 결합

쉼표(,)로 여러 case를 하나로 묶을 수 있습니다.

```java
String day = "토요일";
String type = switch (day) {
    case "월요일", "화요일", "수요일", "목요일", "금요일" -> "평일";
    case "토요일", "일요일" -> "주말";
    default -> "잘못된 요일";
};
System.out.println(type); // 주말
```

### yield 키워드

복잡한 명령이 필요할 때 블록 안에서 yield로 값을 반환합니다.

```java
int score = 90;
String grade = switch (score) {
    case 100, 90 -> {
        System.out.println("훌륭한 성적입니다.");
        yield "A";
    }
    case 80 -> "B";
    case 70 -> "C";
    case 60 -> "D";
    default -> "F";
};
System.out.println("학점: " + grade); // 훌륭한 성적입니다. / 학점: A
```

## 4. 문제 풀이 - 점수에 따른 학점 판별

### 문제

시험 점수(score)에 따른 학점을 출력하는 코드

학점 기준:
- 90점 이상 ~ 100점 이하: A
- 80점 이상 ~ 89점 이하: B
- 70점 이상 ~ 79점 이하: C
- 60점 이상 ~ 69점 이하: D
- 0점 이상 ~ 59점 이하: F
- 그 외의 점수: "올바르지 않은 점수입니다."

### 풀이 (if-else if 사용)

```java
int score = 89;

if (score >= 90 && score <= 100) {
    System.out.println("A");
} else if (score >= 80 && score <= 89) {
    System.out.println("B");
} else if (score >= 70 && score <= 79) {
    System.out.println("C");
} else if (score >= 60 && score <= 69) {
    System.out.println("D");
} else if (score >= 0 && score <= 59) {
    System.out.println("F");
} else {
    System.out.println("올바르지 않은 점수입니다.");
}
```

### 풀이 원리

각 등급의 범위를 체크합니다:
- 가장 높은 등급부터 순차적으로 검사
- 해당하는 범위를 찾으면 출력하고 종료
- 모든 조건이 거짓이면 마지막 else 블록 실행

## 5. switch 표현식 vs if-else if

### if-else if 사용하는 경우

- 범위 비교가 필요할 때 (a >= 90 && a <= 100)
- 복잡한 조건 연산이 필요할 때

위 문제처럼 **범위 비교**가 필요하면 if-else if를 사용합니다.

### switch 표현식 사용하는 경우

- 정확한 값 비교일 때
- 경우의 수가 많을 때
- 코드가 더 깔끔할 때

예: 요일 구분, 권한 체크 등

## 6. 반복문 (오후 수업)

조건식이 참인 동안 코드를 반복해서 실행합니다.

반복문의 종류:
- while: 반복 횟수가 정해지지 않을 때
- do-while: 최소 1회 이상 실행이 필요할 때
- for: 반복 횟수가 명확할 때

## 7. while문

지정한 조건식이 참일 경우 조건이 거짓이 될 때까지 해당 블록을 반복하여 수행합니다.

주로 반복 횟수는 정해져 있지 않지만 특정 조건이 만족할 때까지 루프를 유지할 때 사용합니다.

### 형식

```java
while (조건식) {
    // 조건식이 참인 동안 반복해서 실행할 구문
}
```

### 예시

```java
int i = 0;
while (i < 3) {
    System.out.println("while 루프: " + i);
    i++;
}
```

### 터미널에서 값 입력받기

```bash
java ch08.WhileTest 10
```

프로그램 실행 시 명령줄 인수(argument)로 값을 전달할 수 있습니다.

```java
public class WhileTest {
    public static void main(String[] args) {
        int num = Integer.parseInt(args[0]); // "10" → 10
        int sum = 0;
        int i = 1;
        
        while (i <= num) {
            sum += i;
            i++;
        }
        System.out.println("1부터 " + num + "까지의 합계: " + sum);
    }
}
```

출력: `1부터 10까지의 합계: 55`

### Math.random() 함수

0이상 1미만의 무작위 double 타입의 숫자를 반환합니다.

```java
double num = Math.random();
System.out.println(num); // 0.123456... 같은 랜덤 값

// 1부터 10까지의 랜덤 숫자
int randomNum = (int)(Math.random() * 10) + 1;
```

## 8. do-while문

일단 블록의 내부 구문을 최초 1회 무조건 먼저 실행한 후, 조건식을 사후 검사하여 조건이 참인 동안 루프를 지속하는 제어문입니다.

조건식의 만족 여부에 상관없이 최소 1번의 실행을 보장해야 할 때 주로 사용합니다.

### 형식

```java
do {
    // 최소 1회 무조건 실행 및 조건식이 참인 동안 반복해서 실행할 구문
} while (조건식);
```

### 예시

```java
int j = 10;
do {
    System.out.println("do-while 무조건 한번은 실행됨: " + j);
} while (j < 5); // 거짓이지만 위에서 이미 한번 출력됨
```

출력:
```
do-while 무조건 한번은 실행됨: 10
```

### while vs do-while

- while: 조건을 먼저 확인 후 실행 (0회 실행 가능)
- do-while: 일단 실행 후 조건 확인 (최소 1회 실행)

## 9. for문

지정한 조건식이 참일 경우 조건이 거짓이 될 때까지 해당 블록을 반복하여 수행합니다.

반복 횟수가 명확할 때 주로 사용합니다.

### 형식

```java
for (초기화; 조건식; 증감식) {
    // 조건식이 참인 동안 반복해서 실행할 구문
}
```

### 실행 순서

1. 초기화 실행
2. 조건식 확인
3. 조건식이 true이면 구문 실행
4. 증감식 실행
5. 2번부터 반복 (조건식이 false가 될 때까지)

### 예시

```java
for (int i = 0; i < 5; i++) {
    System.out.println("현재 숫자: " + i); // 0부터 4까지 5번 출력
}
```

출력:
```
현재 숫자: 0
현재 숫자: 1
현재 숫자: 2
현재 숫자: 3
현재 숫자: 4
```

### for문의 각 부분

- **초기화**: 루프가 시작될 때 1회만 실행 (int i = 0)
- **조건식**: 매 루프마다 확인 (i < 5)
- **증감식**: 매 루프마다 실행 (i++)
