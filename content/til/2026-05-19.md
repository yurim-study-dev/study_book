## 1. 변수와 상수

### 변수 (Variable)
데이터를 담는 그릇

- 변수에 값을 저장한 후 필요할 때 사용
- 언제든지 값을 변경할 수 있음
- 고유한 이름으로 식별됨

### 변수 사용 방법

```java
// 1. 변수 선언만
int age;

// 2. 변수에 값 할당
age = 25;

// 3. 선언과 동시에 초기값 부여
int score = 100;

// 변수는 값을 변경할 수 있음
age = 26;
```

### 다양한 데이터 타입

```java
byte b = 100;              // 작은 정수
short s = 10000;           // 중간 정수
int num = 100000000;       // 일반 정수
long bigNum = 3000000000L; // 큰 정수
double pi = 3.14159;       // 실수
float smallPi = 3.14F;     // 작은 실수
boolean isGood = true;     // 참/거짓
char grade = 'A';          // 한 글자
String name = "멋쟁이";    // 문자열
```

### 상수 (Constant)
한 번 값이 정해지면 변경 불가능

```java
final int MAX_SIZE = 100;   // 상수 선언
// MAX_SIZE = 200;  // 에러! 상수는 변경 불가능

final double PI = 3.14159;
final int MIN_AGE = 18;
```

상수의 관례:
- 모두 대문자로 작성
- 단어 사이는 언더스코어(_)로 구분

### 변수 vs 상수

| 구분 | 변수 | 상수 |
|------|------|------|
| 값 변경 | 가능 | 불가능 |
| 선언 | `int a;` | `final int A;` |
| 이름 규칙 | 소문자 시작 | 대문자 |

## 3. 식별자와 예약어

### 식별자 명명 규칙

- 영문, 숫자, `_`, `$`만 사용 가능
- 숫자로 시작 불가능
- 예약어는 사용 불가능

### 식별자 명명 관례

```java
int age;              // 변수: 소문자로 시작
int myScore;          // 여러 단어는 두 번째부터 대문자

class Student { }     // 클래스: 대문자로 시작

final int MAX_SIZE;   // 상수: 모두 대문자
```

### 예약어

int, double, class, public, if, for, true, false, final 등
이들은 변수명으로 사용할 수 없음

## 4. 형변환 (Type Casting)

서로 다른 데이터 타입을 변환하는 것

### 자동 형변환 (작은 → 큰)

데이터 손실 없음

```
byte → short → int → long → float → double
```

```java
int a = 10;
double b = a;  // int → double 자동 변환
System.out.println(b);  // 10.0
```

### 강제 형변환 (큰 → 작은)

데이터 손실 가능

```java
double pi = 3.14;
int p = (int)pi;  // double → int 강제 변환 (괄호 필수)
System.out.println(p);  // 3 (소수점 버려짐)
```

### INT → DOUBLE vs DOUBLE → INT

- INT → DOUBLE: 손실 없음
- DOUBLE → INT: 소수점이 버려짐

## 5. 연산자

### 산술 연산자

```java
int result1 = 5 + 3;   // 더하기
int result2 = 5 - 3;   // 빼기
int result3 = 5 * 3;   // 곱하기
int result4 = 5 / 2;   // 나누기 (정수 나눗셈)
int result5 = 5 % 2;   // 나머지

// 문자열과 함께 사용하면 연결
String message = "나이는 " + 25 + "살입니다.";
```

### 대입 연산자

```java
int a = 10;      // a에 10 대입
a += 5;          // a = a + 5
a -= 3;          // a = a - 3
a *= 2;          // a = a * 2
a /= 4;          // a = a / 4
a %= 3;          // a = a % 3
```

### 증감 연산자

```java
int count = 10;
count++;  // 1씩 증가
count--;  // 1씩 감소
```

#### 전위 vs 후위 연산

```java
// 전위 연산 (++a): 먼저 증가, 그 다음 사용
int a = 10;
int b = ++a;  // a를 먼저 11로 증가
System.out.println("a: " + a + ", b: " + b);  // a: 11, b: 11

// 후위 연산 (a++): 먼저 사용, 그 다음 증가
int x = 10;
int y = x++;  // x의 현재값 10을 y에 먼저 대입
System.out.println("x: " + x + ", y: " + y);  // x: 11, y: 10
```

### 비교 연산자

```java
System.out.println(10 == 10);  // true (같다)
System.out.println(10 != 5);   // true (다르다)
System.out.println(10 > 5);    // true (크다)
System.out.println(10 < 5);    // false (작다)
System.out.println(10 >= 10);  // true (크거나 같다)
System.out.println(10 <= 5);   // false (작거나 같다)
```

### 논리 연산자

#### && (AND) - 둘 다 참이어야 참

```java
int age = 19;
boolean isTeenage = (age >= 10) && (age < 20);
System.out.println(isTeenage);  // true
```

#### || (OR) - 하나만 참이어도 참

```java
int age = 8;
boolean isFreeRide = (age <= 8) || (age >= 65);
System.out.println(isFreeRide);  // true
```

#### ! (NOT) - 반대 의미

```java
boolean isRainy = true;
boolean isSunny = !isRainy;
System.out.println(isSunny);  // false
```

### 삼항 연산자

조건에 따라 두 값 중 하나 선택

```java
int score = 85;
String result = (score >= 80) ? "합격" : "불합격";
System.out.println(result);  // 합격
```

## 6. 연산자 우선순위

여러 연산자가 섞여있을 때 실행 순서

| 순위 | 연산자 유형 | 기호 | 방향 |
|------|----------|------|------|
| 1 | 괄호 | `()` | 왼쪽→오른쪽 |
| 2 | 단항 | `++`, `--`, `+`, `-`, `!` | 오른쪽→왼쪽 |
| 3 | 곱셈 계열 | `*`, `/`, `%` | 왼쪽→오른쪽 |
| 4 | 덧셈 계열 | `+`, `-` | 왼쪽→오른쪽 |
| 5 | 비교 (크기) | `<`, `<=`, `>`, `>=` | 왼쪽→오른쪽 |
| 6 | 비교 (동등) | `==`, `!=` | 왼쪽→오른쪽 |
| 7 | 논리 AND | `&&` | 왼쪽→오른쪽 |
| 8 | 논리 OR | `\|\|` | 왼쪽→오른쪽 |
| 9 | 삼항 | `? :` | 오른쪽→왼쪽 |
| 10 | 대입 | `=`, `+=` 등 | 오른쪽→왼쪽 |

### 실제 예시

```java
// 곱셈이 덧셈보다 먼저
int result1 = 5 + 3 * 2;  // 3 * 2 = 6, 5 + 6 = 11
System.out.println(result1);  // 11

// 괄호가 최우선
int result2 = (5 + 3) * 2;  // 5 + 3 = 8, 8 * 2 = 16
System.out.println(result2);  // 16

// 비교가 논리보다 먼저
boolean result3 = 10 > 5 && 3 < 1;  // true && false = false
System.out.println(result3);  // false

// 헷갈릴 때는 괄호 사용
int result4 = (5 + 3) * (2 + 1);  // 8 * 3 = 24
System.out.println(result4);  // 24
```

## 핵심 요점

- 변수: 값을 담는 그릇, 변경 가능
- 상수: final 키워드 사용, 변경 불가능
- INT: 정수, DOUBLE: 실수
- 형변환: 작은→큰은 자동, 큰→작은은 강제
- 연산자 우선순위: 괄호 → 단항 → 곱셈/나눗셈 → 덧셈/뺄셈 → 비교 → 논리 → 대입

---

## 배운점

1. 정수끼리 나누면 소수점이 버려지기 때문에 형변환이 필요하다.
2. % 연산자로 나머지를 구할 수 있다.
3. 삼항 연산자는 조건에 따라 서로 다른 값을 반환하는 간단한 방법이다.
4. 논리 연산자 &&와 ||을 조합하면 복잡한 조건도 표현할 수 있다.
5. 연산자 우선순위를 알면 코드의 의도를 정확히 파악할 수 있다.
6. 변수는 언제든 값을 변경할 수 있지만, 상수는 한 번 정해지면 변경할 수 없다.
7. 각 데이터 타입마다 표현할 수 있는 범위가 다르다.

### [문제 3] 윤년 판별

```java
boolean isLeapYear = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
```

윤년 규칙:
- 4의 배수이면서 100의 배수가 아니면 윤년 (2024, 2020)
- 400의 배수이면 윤년 (2000)
- 100의 배수이지만 400의 배수가 아니면 평년 (1900)

예시:
- 2024: (2024%4==0 && 2024%100!=0) → true → 윤년
- 1900: (1900%4==0 && 1900%100!=0) → false, (1900%400==0) → false → 평년
- 2000: (2000%4==0 && 2000%100!=0) → false, (2000%400==0) → true → 윤년
