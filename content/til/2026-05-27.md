## 1. 2차원 배열 (다중행 배열)

### 2차원 배열이란?

배열 안에 다시 배열이 있는 구조입니다.

행(row)과 열(column)로 이루어진 표 형태의 자료구조입니다.

### 2차원 배열 선언 및 생성

```java
// 2행 3열의 2차원 배열 생성
int[][] matrix = new int[2][3];

// 배열 요소에 접근
matrix[0][1] = 10;  // 0행 1열에 10 저장
matrix[1][2] = 20;  // 1행 2열에 20 저장
```

메모리 구조:
```
            열[0]  열[1]  열[2]
            ┌────┬────┬────┐
행[0]  ────►│  0 │ 10 │  0 │
            └────┴────┴────┘
            ┌────┬────┬────┐
행[1]  ────►│  0 │  0 │ 20 │
            └────┴────┴────┘
```

### 2차원 배열 초기화

```java
// 초기화와 동시에 생성
int[][] matrix2 = {
    {10, 20, 30},
    {40, 50},          // 각 행의 길이가 다를 수 있음 (가변 배열)
    {70, 80, 90}
};
```

### 2차원 배열 순회 (이중 for문)

```java
int[][] matrix2 = {
    {10, 20, 30},
    {40, 50},
    {70, 80, 90}
};

// 모든 요소 출력
for (int i = 0; i < matrix2.length; i++) {           // 행 순회
    for (int k = 0; k < matrix2[i].length; k++) {    // 열 순회
        System.out.println("행: " + i + ", 열: " + k);
        System.out.println(matrix2[i][k]);
    }
}
```

**순회 과정:**
- 외부 for: i = 0, 1, 2 (행)
- 내부 for: k = 0, 1, 2... (각 행의 열)
- matrix2[i].length: i번째 행의 길이

## 2. Enhanced for 루프 (향상된 for문)

### Enhanced for란?

배열의 모든 요소를 순회할 때 인덱스 없이 직접 요소에 접근하는 방법입니다.

### 기본 형식

```java
for (타입 변수명 : 배열명) {
    // 배열의 각 요소가 변수명에 대입됨
}
```

### 일반 for vs Enhanced for

**일반 for문:**
```java
int[] scores = {100, 80, 70, 88};

int sum = 0;
for (int i = 0; i < scores.length; i++) {
    sum += scores[i];  // 인덱스로 접근
}
```

**Enhanced for문:**
```java
int[] scores = {100, 80, 70, 88};

int sum = 0;
for (int score : scores) {  // 요소를 직접 사용
    sum += score;
}
```

### 장점

- 코드가 더 간결하고 읽기 쉬움
- 인덱스 관리 불필요
- 배열의 길이가 변해도 코드 수정 불필요

### 2차원 배열과 Enhanced for

```java
int[][] matrix = {
    {10, 20, 30},
    {40, 50, 60}
};

// 각 행을 순회
for (int[] row : matrix) {           // 각 행을 row에 대입
    for (int value : row) {          // 각 행의 요소를 순회
        System.out.println(value);
    }
}
```

## 3. 배열 복사

### 3-1. 얕은 복사 (Shallow Copy)

배열의 주소값만 복사하는 방식입니다.

```java
int[] original = {10, 20, 30};
int[] shallow = original;  // 주소값만 복사

shallow[0] = 99;  // 값 변경

// 결과
System.out.println("original[0]: " + original[0]);  // 99
System.out.println("shallow[0]: " + shallow[0]);    // 99
```

**메모리 구조:**
```
스택:
┌──────────────┐
│ original ────┼─────┐
│ shallow ─────┼──┐  │
└──────────────┘  │  │
                  │  │
힙:              │  │
                  │  │
     ┌────┬────┬────┐
     │ 99 │ 20 │ 30 │◄──┴──┘ (같은 배열)
     └────┴────┴────┘
```

**문제점:** 원본과 복사본이 같은 배열을 가리키므로 한쪽을 수정하면 다른 쪽도 영향을 받음

---

### 3-2. 깊은 복사 (Deep Copy)

실제 배열의 모든 요소를 새로운 배열에 복사하는 방식입니다.

#### 방법 1: For 루프를 사용한 복사

```java
int[] original = {10, 20, 30};
int[] deep = new int[original.length];  // 새로운 배열 생성

// 원본 배열의 각 요소를 복사
for (int i = 0; i < original.length; i++) {
    deep[i] = original[i];
}

deep[0] = 99;  // 복사본 수정

// 결과
System.out.println("original[0]: " + original[0]);  // 10
System.out.println("deep[0]: " + deep[0]);          // 99
```

#### 방법 2: System.arraycopy() 사용

```java
int[] original = {10, 20, 30};
int[] deep = new int[original.length];

// System.arraycopy(원본, 원본시작인덱스, 복사본, 복사본시작인덱스, 복사개수)
System.arraycopy(original, 0, deep, 0, original.length);

deep[0] = 99;  // 복사본 수정

// 결과
System.out.println("original[0]: " + original[0]);  // 10
System.out.println("deep[0]: " + deep[0]);          // 99
```

#### 방법 3: Arrays.copyOf() 사용 (가장 간단)

```java
import java.util.Arrays;

int[] original = {10, 20, 30};
int[] deep = Arrays.copyOf(original, original.length);

deep[0] = 99;  // 복사본 수정

// 결과
System.out.println("original[0]: " + original[0]);  // 10
System.out.println("deep[0]: " + deep[0]);          // 99
```

**메모리 구조:**
```
스택:
┌──────────────┐
│ original ────┼───┐
│ deep ────────┼─┐ │
└──────────────┘ │ │
                 │ │
힙:              │ │
     ┌────┬────┬────┐
     │ 10 │ 20 │ 30 │◄─┘ (원본)
     └────┴────┴────┘

     ┌────┬────┬────┐
     │ 99 │ 20 │ 30 │◄── (복사본, 독립적)
     └────┴────┴────┘
```

### 얕은 복사 vs 깊은 복사

| 특징 | 얕은 복사 | 깊은 복사 |
|------|---------|---------|
| 주소 | 같음 | 다름 |
| 메모리 | 추가 할당 안 함 | 새로운 배열 생성 |
| 수정 영향 | 원본도 영향 | 원본에 영향 없음 |
| 사용 시기 | 드물게 | 대부분의 경우 |

## 4. 배열 관련 메서드

### Arrays 클래스의 유용한 메서드

```java
import java.util.Arrays;

// 배열 복사
int[] copy = Arrays.copyOf(original, original.length);

// 배열 일부 복사
int[] partial = Arrays.copyOfRange(original, 1, 3);  // 인덱스 1부터 2까지

// 배열 출력
System.out.println(Arrays.toString(original));  // [10, 20, 30]

// 배열 정렬
Arrays.sort(original);

// 배열 검색
int index = Arrays.binarySearch(original, 20);
```

---

## 배운점

1. 2차원 배열은 행(row)과 열(column)로 이루어진 표 형태이다.
2. 2차원 배열은 이중 for문으로 순회한다.
3. Enhanced for는 배열의 모든 요소를 순회할 때 인덱스 없이 접근한다.
4. 얕은 복사는 주소값만 복사하므로 원본에 영향을 받는다.
5. 깊은 복사는 실제 요소들을 새로운 배열에 복사한다.
6. 깊은 복사 방법은 3가지가 있다: for문, System.arraycopy(), Arrays.copyOf()
7. Arrays.copyOf()가 가장 간결하고 효율적이다.
8. 대부분의 경우 깊은 복사를 사용해야 원본 데이터를 보호할 수 있다.

## 코드 정리

### 2차원 배열 순회
```java
int[][] matrix = {{10, 20}, {30, 40}};

// 일반 for문
for (int i = 0; i < matrix.length; i++) {
    for (int k = 0; k < matrix[i].length; k++) {
        System.out.println(matrix[i][k]);
    }
}

// Enhanced for
for (int[] row : matrix) {
    for (int value : row) {
        System.out.println(value);
    }
}
```

### 배열 깊은 복사
```java
import java.util.Arrays;

int[] original = {10, 20, 30};

// 방법 1: Arrays.copyOf() (권장)
int[] copy = Arrays.copyOf(original, original.length);

// 방법 2: System.arraycopy()
int[] copy2 = new int[original.length];
System.arraycopy(original, 0, copy2, 0, original.length);

// 방법 3: for문
int[] copy3 = new int[original.length];
for (int i = 0; i < original.length; i++) {
    copy3[i] = original[i];
}
```
