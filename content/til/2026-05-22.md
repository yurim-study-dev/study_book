## 1. 오전 - FOR 문 실습

### 문제 1: 1~100 합계 관련 문제

#### 문제 1-1: 1~100까지의 합계

```java
int totalSum = 0;
for (int num = 1; num <= 100; num++) {
    totalSum += num;
}
System.out.println("1~100까지의 합계: " + totalSum); // 5050
```

풀이: 1부터 100까지 모두 더함

#### 문제 1-2: 1~100 홀수 합계

```java
int oddSum = 0;
for (int num = 1; num <= 100; num += 2) { // 1, 3, 5, 7 ...
    oddSum += num;
}
System.out.println("1~100까지 홀수 합계: " + oddSum); // 2500
```

풀이: num += 2로 홀수만 반복 (1, 3, 5, 7, ...)

#### 문제 1-3: 1~100 짝수 합계

```java
int evenSum = 0;
for (int num = 2; num <= 100; num += 2) { // 2, 4, 6, 8 ...
    evenSum += num;
}
System.out.println("1~100까지 짝수 합계: " + evenSum); // 2550
```

풀이: num = 2에서 시작, num += 2로 짝수만 반복 (2, 4, 6, 8, ...)

## 2. 계산기 프로그램 (CalculatorTest.java)

### Scanner를 사용한 입력받기

사용자로부터 데이터를 입력받을 수 있습니다.

```java
import java.util.Scanner;

Scanner s = new Scanner(System.in);

// 정수 입력받기
int num1 = s.nextInt();

// 문자열 입력받기
String operator = s.next();

// 또 다른 정수 입력받기
int num2 = s.nextInt();
```

### 계산기 프로그램 구조

```java
import java.util.Scanner;

public class CalculatorTest {
    public static void main(String[] args) {
        System.out.println("*** 계산기 프로그램 ***");
        System.out.println("계산식을 입력하세요.(예시, 5+4)");
        
        Scanner s = new Scanner(System.in);
        
        System.out.printf("첫번째 숫자를 입력하세요 : ");
        int num1 = s.nextInt();
        
        System.out.printf("연산자를 입력하세요 ( 보기 : + , - , * , / ) : ");
        String operator = s.next();
        
        System.out.printf("두번째 숫자를 입력하세요 : ");
        int num2 = s.nextInt();
        
        String result = switch (operator) {
            case "+" -> num1 + " " + operator + " " + num2 + " = " + (num1 + num2);
            case "-" -> num1 + " " + operator + " " + num2 + " = " + (num1 - num2);
            case "*" -> num1 + " " + operator + " " + num2 + " = " + (num1 * num2);
            case "/" -> num1 + " " + operator + " " + num2 + " = " + ((double)num1 / num2);
            default -> "수식에 오류가 있습니다.";
        };
        
        System.out.println(result);
    }
}
```

### 주요 포인트

- `Scanner`: 사용자 입력을 받는 객체
- `s.nextInt()`: 정수 입력받기
- `s.next()`: 문자열 입력받기
- 나누기(`/`)는 `(double)`로 형변환하여 소수점이 나오도록 함

실습: `ch08/CalculatorTest.java`

## 3. 오후 - 이중 중첩 for문으로 별 만들기

### 이중 중첩 for문

for문 안에 또 다른 for문이 있는 구조입니다.

외부 for문: 행(row) 제어
내부 for문: 열(column) 제어

### 패턴 1: 증가하는 별

```java
// *
// **
// ***
// ****
// *****

static void star2() {
    for (int i = 0; i < 5; i++) {
        for (int k = 0; k < i + 1; k++) {
            System.out.print("*");
        }
        System.out.println();
    }
}
```

실행 과정:
- i=0: k는 0~0 반복 → * (1개)
- i=1: k는 0~1 반복 → ** (2개)
- i=2: k는 0~2 반복 → *** (3개)
- i=3: k는 0~3 반복 → **** (4개)
- i=4: k는 0~4 반복 → ***** (5개)

### 패턴 2: 감소하는 별

```java
// *****
// ****
// ***
// **
// *

static void star3() {
    for (int i = 0; i < 5; i++) {
        for (int k = 0; k < 5 - i; k++) {
            System.out.print("*");
        }
        System.out.println();
    }
}
```

실행 과정:
- i=0: k는 0~4 반복 → ***** (5개)
- i=1: k는 0~3 반복 → **** (4개)
- i=2: k는 0~2 반복 → *** (3개)
- i=3: k는 0~1 반복 → ** (2개)
- i=4: k는 0~0 반복 → * (1개)

### 패턴 3: 피라미드 별

```java
//     *      (스페이스 4개, 별 1개)
//    **      (스페이스 3개, 별 2개)
//   ***      (스페이스 2개, 별 3개)
//  ****      (스페이스 1개, 별 4개)
// *****      (스페이스 0개, 별 5개)

static void star4() {
    for (int i = 0; i < 5; i++) {
        // 스페이스 출력하는 부분
        for (int k = 0; k < 4 - i; k++) {
            System.out.print(" ");
        }
        // 별을 출력하는 부분
        for (int k = 0; k < i + 1; k++) {
            System.out.print("*");
        }
        System.out.println();
    }
}
```

실행 과정:
- i=0: 스페이스 4개, 별 1개
- i=1: 스페이스 3개, 별 2개
- i=2: 스페이스 2개, 별 3개
- i=3: 스페이스 1개, 별 4개
- i=4: 스페이스 0개, 별 5개

### 구구단 (ForTest6.java)

```java
// 2 * 1 = 2  3 * 1 = 3  4 * 1 = 4 ... 9 * 1 = 9
// 2 * 2 = 4  3 * 2 = 6  4 * 2 = 8 ... 9 * 2 = 18
// ...
// 2 * 9 = 18  3 * 9 = 27  4 * 9 = 36 ... 9 * 9 = 81

for (int i = 1; i <= 9; i++) {      // 행 (9행)
    for (int k = 2; k <= 9; k++) {  // 열 (8열)
        System.out.print(k + " * " + i + " = " + (k * i) + " ");
    }
    System.out.println();
}
```

외부 for: 1~9 (곱해질 수)
내부 for: 2~9 (곱하는 수)

실습: `ch08/ForTest5.java`, `ch08/ForTest6.java`

## 4. 메서드 (Method)

### 메서드란?

같은 기능의 코드를 여러 번 사용할 때, 메서드로 만들어 호출하면 됩니다.

### 메서드 형식

```java
static 반환타입 메서드명() {
    // 실행할 코드
}
```

### 메서드 호출

```java
// 메서드 정의
static void star1() {
    System.out.println("별 출력");
}

// 메서드 호출
public static void main(String[] args) {
    star1();  // 실행
}
```

## 5. continue와 break 문

루프 제어문으로 반복의 흐름을 변경합니다.

### continue 문

현재 반복을 건너뛰고 다음 반복으로 넘어갑니다.

```java
// continue를 이용한 홀수 합계
int sum = 0;
for (int i = 1; i <= 100; i++) {
    if (i % 2 == 0) { // 짝수일 때
        continue;      // 이 반복을 건너뜀
    }
    sum += i;
}
System.out.println("1~100 홀수 합계: " + sum); // 2500
```

실행 과정:
- i=1: 홀수 → sum에 추가
- i=2: 짝수 → continue로 건너뜀
- i=3: 홀수 → sum에 추가
- i=4: 짝수 → continue로 건너뜀
- ...

### break 문

루프를 즉시 종료합니다.

```java
int num = 1;
int sum = 0;
while (num <= 100) {
    sum += num;
    num++;
    
    if (num > 100) {
        break; // while 루프를 즉시 종료
    }
}
System.out.println("1~100 합계: " + sum); // 5050
```

실행 과정:
- num이 101이 되면 조건을 확인하고 break 실행
- while 루프 종료

### continue vs break

| 구분 | continue | break |
|------|----------|-------|
| 역할 | 현재 반복 건너뜀 | 루프 종료 |
| 다음 반복 | 계속 진행 | 루프 종료 |
| 사용처 | 특정 조건 제외 | 루프 탈출 조건 |

## 6. 과제 - 게임 프로그램 (homework.h01)

### 게임의 개요

플레이어가 적과 전투하는 간단한 게임입니다.

요소:
- 플레이어 HP: 초기값 50~100
- 적 HP: 10~99 (랜덤)
- 행동 선택: 싸우기, 도망가기, 회복, 필살기
- 승리/패배 조건

### 주요 코드 구조

```java
import java.util.Scanner;

public class Game {
    public static void main(String[] args) {
        Scanner s = new Scanner(System.in);
        
        // 변수 선언
        String name = s.next();
        int hp = 50;
        int enemyHp = (int)(Math.random() * 90) + 10;
        
        // 무한 루프
        while (true) {
            // 행동 선택
            int action = s.nextInt();
            
            // 행동에 따른 처리
            switch (action) {
                case 1 -> { /* 싸우기 */ }
                case 2 -> { /* 도망가기 */ }
                case 3 -> { /* 회복 */ }
                case 4 -> { /* 필살기 */ }
            }
            
            // 게임 종료 조건
            if (hp <= 0 || enemyHp <= 0) {
                break;
            }
        }
        
        // 승패 결정
        if (hp <= 0) {
            System.out.println("당신은 사망했습니다.");
        } else if (enemyHp <= 0) {
            System.out.println("전투에 승리했습니다.");
        }
    }
}
```

### 게임 플레이 로직

1. **싸우기 (case 1)**
   - 플레이어가 적에게 0~10 데미지
   - 적이 플레이어에게 0~10 데미지

2. **도망가기 (case 2)**
   - 플레이어가 0~3 데미지 입음 (도망치는 데 비용)
   - 새로운 적 등장 (10~99 HP)

3. **회복 (case 3)**
   - HP 10 회복

4. **필살기 (case 4)**
   - 필살기 횟수 1 감소
   - 적을 즉시 처치하거나 큰 데미지

### 핵심 개념

```java
// while(true): 무한 루프
while (true) {
    // 처리...
    
    // break로 탈출
    if (hp <= 0 || enemyHp <= 0) {
        break;
    }
}
```
